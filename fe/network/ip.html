                    # IP

                    # IP协议

                    IP协议是TCP/IP协议族中最核心的协议，它负责将数据包从源主机传输到目标主机。

                    IP协议的主要特性：

                    • IP协议是一种无连接的、不可靠的协议，它不会对数据包进行任何形式的检查和确认，也不会对数据包进行重传，因此它不能保证数据包的可靠传输。

                    • IP协议是一种无状态的协议，它不会对数据包进行任何形式的状态记录，也不会对数据包进行任何形式的状态检查，因此它不能保证数据包的顺序传输。

                    # IP地址

                    IP地址是IP协议中用于标识主机的一种地址，它由32位二进制数组成，通常以点分十进制的形式表示。

                    • D类地址：以1110开头，范围：
                    • E类地址：以1111开头，范围：

                    IP地址的分类主要用于划分网络的范围，不同的IP地址范围对应不同的网络范围。

                    IP地址的划分：

                    • 网络地址：IP地址中网络部分的部分，用于标识网络。
                    • 主机地址：IP地址中主机部分的部分，用于标识主机。

                    # IP数据包

                    IP数据包是IP协议中用于传输数据的一种数据包，它由IP头部和IP数据组成。

                    IP数据包的主要特性：

                    • IP数据包的最大长度为65535字节，其中IP头部的长度为20字节，IP数据的最大长度为65515字节。

                    • IP数据包的传输是不可靠的，它不会对数据包进行任何形式的检查和确认，也不会对数据包进行重传，因此它不能保证数据包的可靠传输。

                    • IP数据包的传输是无状态的，它不会对数据包进行任何形式的状态记录，也不会对数据包进行任何形式的状态检查，因此它不能保证数据包的顺序传输。
